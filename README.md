# 甜菜答案词 SweetBeet English

一个英语单词听写练习网页应用，具有可爱的绘本风格和物理引擎橘子奖励系统。

## 功能特点

- 📱 **响应式设计**: 完美适配电脑端和手机端
- 🔐 **白名单验证**: 只有购买用户才能使用
- 🎵 **音频播放**: 播放单词发音进行听写练习
- 💡 **智能提示**: 可查看单词中文意思
- 🍊 **物理奖励**: 答对获得掉落橘子，具有真实物理效果
- 🎨 **治愈系UI**: 柔和色彩，绘本风格，微动画效果

## 使用说明

### 1. 登录
- 输入手机号进行验证
- 只有白名单中的手机号才能进入学习

### 2. 学习流程
- 点击播放按钮听单词发音
- 在输入框中输入听到的单词拼写
- 点击确认按钮提交答案
- 如需帮助可点击提示按钮查看中文意思

### 3. 奖励机制
- 不使用提示答对：获得橘子奖励 🍊
- 使用提示后答对：不获得橘子
- 答错：输入框抖动提示，继续当前单词

### 4. 完成庆祝
- 答完所有单词后橘子雨庆祝效果
- 显示最终获得的橘子数量

## 技术实现

- **前端**: HTML5 + CSS3 + JavaScript
- **物理引擎**: Matter.js
- **响应式**: CSS媒体查询
- **动画**: CSS动画 + Canvas绘制

## 自定义配置

### 修改白名单
在 `script.js` 中修改 `whitelist` 数组：
```javascript
const whitelist = [
    '13800138000',
    '13900139000',
    // 添加更多手机号...
];
```

### 修改单词数据
在 `script.js` 中修改 `wordsData` 数组：
```javascript
const wordsData = [
    { words: ["单词"], hint: "中文意思", audio: "音频文件路径" },
    // 添加更多单词...
];
```

### 添加音频文件
1. 创建 `audio` 文件夹
2. 将对应的 mp3 文件放入其中
3. 确保文件名与 `wordsData` 中的路径匹配
